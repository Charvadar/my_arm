This is my custom Armenian Phonetic keyboard for MACOS.
Differences from original one:
1. SHIFT + / = ? (insteag of ՞)
2. SHIFT + 9 = ( (insteag of Չ)
3. SHIFT + 0 = ) (insteag of Ճ)
4. SHIFT + ; = : (insteag of .)
5. SHIFT + , = < (insteag of «)
6. SHIFT + . = > (insteag of »)
